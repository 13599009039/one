# Template Directory

This directory contains extracted modal templates for dynamic loading.

## Structure
- modal-order-detail.html: 订单详情模态框
- modal-order-add.html: 新增订单模态框  
- modal-customer-edit.html: 客户编辑模态框
- modal-task-detail.html: 任务详情模态框
- modal-inventory.html: 库存出入库模态框
